Gráfico I — Limites

LC = x̄
LCS = x̄ + 3 × (MR̄ / d₂)
LCI = x̄ − 3 × (MR̄ / d₂)
d₂ = 1,128  (para n = 2)

Gráfico MR — Limites

MRᵢ = |xᵢ − xᵢ₋₁|
LC = MR̄
LCS = D₄ × MR̄
LCI = 0 (convenção)
D₄ = 3,267  (para n = 2)

Quando Usar

• Subgrupo de tamanho n = 1
• Medições individuais no tempo
• Processos lentos ou de alto custo
• Dados administrativos em série
