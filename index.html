Transformação Box-Cox

y(λ) = (xᵏ − 1) / λ , λ ≠ 0
y(λ) = ln(x) , λ = 0
Todos os x devem ser > 0

Lambdas comuns

λ = −1 → inverso (1/x)
λ = −0,5 → inverso raiz
λ = 0 → logaritmo natural
λ = 0,5 → raiz quadrada
λ = 1 → sem transformação
λ = 2 → quadrado

Quando usar

• Dados rejeitam normalidade (A-D)
• Histograma assimétrico
• Antes de calcular Cp/Cpk
• Dados de tempo, custo, contagem

Método

Maximiza log-verossimilhança
L(λ) = −n/2 · ln(Var(y(λ)))
+ (λ−1) · Σln(xᵢ)
λ ótimo = argmax L(λ)

Cp e Cpk (within)

Cp = (LSE−LIE) / (6σ_w)
Cpk = min[(LSE−x̄), (x̄−LIE)] / (3σ_w)
σ_w = MR̄/d₂ (baseado na variação interna)

Pp e Ppk (overall)

Pp = (LSE−LIE) / (6σ_o)
Ppk = min[(LSE−x̄), (x̄−LIE)] / (3σ_o)
σ_o = desvio padrão amostral (n−1)

Sigma Level e PPM

σ_level = Cpk × 3
PPM = 1000000 × P(fora espec.)
P via distribuição normal acumulada

Referência

Cp/Cpk ≥ 1,33 → Capaz ✓
1,00 ≤ Cp/Cpk < 1,33 → Marginal ⚠
Cp/Cpk < 1,00 → Incapaz ✗
Cp ≫ Cpk → Processo descentrado

Correlação de Pearson

r = Σ(xᵢ−x̄)(yᵢ−ȳ) /
    √[Σ(xᵢ−x̄)²·Σ(yᵢ−ȳ)²]
−1 ≤ r ≤ 1

Regressão Linear

ŷ = b₀ + b₁·x
b₁ = r·(sᵧ/sₓ)
b₀ = ȳ − b₁·x̄

Interpretação de r

|r| ≥ 0,90 → Muito forte
0,70 ≤ |r| < 0,90 → Forte
0,50 ≤ |r| < 0,70 → Moderado
0,30 ≤ |r| < 0,50 → Fraco
|r| < 0,30 → Muito fraco
